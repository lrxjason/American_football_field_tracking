# American_football_field_tracking

1. copy the video to the folder  "Video"

2. change the video name without the extended name.   like "Rudy Sanchez - CFB ALL 22 1"

3. You need to mark the 4 initial points between the frame and the court image. Using the right side of the court for camera end-view.

4. select the tracking area on the first frame

4. The result are saved in the result folder
